## Descripción breve del proyecto
Este proyecto consiste en el desarrollo de un sitio web institucional para el Instituto Politécnico Formosa, con páginas de inicio, institucional, carreras y contacto. Se implementó una estructura responsive y una interfaz clara para presentar la información de la institución.

## Tecnología utilizada
- HTML5
- CSS3
- Bootstrap 5
- Visual Studio Code
